# **README.md — Gabriela Velasco | Art × Code × Movement**

## **Über das Projekt**
Diese Website ist ein hybrides Portfolio, das die künstlerische und technische Identität von **Gabriela Velasco** vereint.  
Es kombiniert:

- zeitgenössische Tanzkunst  
- professionelle Web- & Softwareentwicklung  
- internationale Projekterfahrung  
- IT-Systemadministration  
- IoT-Experimente  
- kreative Medienarbeit  

Die Seite ist vollständig handcodiert in HTML, CSS und JavaScript und nutzt ausschließlich **eigene Fotografien** von Gabriela, um Authentizität und Persönlichkeit zu bewahren.  
Nur im Bereich „Dance Classes“ werden neutrale, offene Silhouetten verwendet.

---

## **Konzept**
Die Website folgt dem Prinzip **Art × Code × Movement**:

- **Art** → Tanz, Choreografie, Körperbewusstsein  
- **Code** → Webentwicklung, IT, IoT, digitale Systeme  
- **Movement** → künstlerische Dynamik, Animationen, Interaktivität  

Sie ist inspiriert von Gabriela’s bestehenden Projekten:

- [Web‑Profile](https://gabivelasco.github.io/Web-Profile/)  
- [ArtisticDance_Web](https://gabivelasco.github.io/ArtisticDance_Web/)  

und vereint beide Stile zu einem **Hybrid‑Design**.
